### REQ-010: Correct document language in scaffolded templates (`lang:` instead of `book.language`)

- **Status:** DONE
- **Added:** 2026-04-26
- **Completed:** 2026-04-26
- **Verified by:** implementation + `matctl course add --help` / `matctl doc add --help`
- **Depends on:** REQ-005
- **Description:**

  The course template ships `book.language: en` in `_quarto.yml`. This key
  is *not* Pandoc/Quarto's document language — it is a custom-language-file
  override. The actual language switch is `lang:` at project or format level.
  The result, observed in `schutzkonzept-ev-christuskirche` (German doc
  scaffolded with the course template), was that crossref labels rendered
  as "Chapter X" instead of "Kapitel X".

  Fix the template, and add a required `--lang {de,en}` flag to
  `course add` / `doc add` so the scaffolder stamps the correct value
  without a manual post-edit. The flag has no default — authors must make
  the language choice explicit at creation time.

- **Acceptance criteria:**
  - [ ] `templates/course/_quarto.yml` has `lang: {{LANG}}` at top level; the `book.language` key is removed.
  - [ ] `templates/doc/_quarto.yml` has `lang: {{LANG}}` at top level.
  - [ ] `matctl course add` accepts `--lang {de,en}` and errors out with a clear message when `--lang` is omitted; `{{LANG}}` substitution is applied during scaffolding.
  - [ ] `matctl doc add` accepts `--lang {de,en}` and errors out with a clear message when `--lang` is omitted; `{{LANG}}` substitution is applied during scaffolding.
  - [ ] A scaffolded German project renders `@sec-…` cross-references as "Kapitel X" / "Abschnitt X" (visual check on a minimal book with two chapters).
  - [ ] `docs/authoring.md` mentions the `lang:` key under §1 (Front matter) with the German vs English consequence on crossref labels.
- **Notes:**

  Source: `material/material-core-input.md` §1; decision 2026-04-26: no
  default language. Existing projects without a `lang:` key continue to
  render as before (Quarto's implicit default); this change only affects
  newly scaffolded projects.
